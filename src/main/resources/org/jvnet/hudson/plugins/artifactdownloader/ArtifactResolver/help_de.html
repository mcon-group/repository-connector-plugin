Definieren sie die Artefakte, die sie herunterladen möchten.
Das Zielverzeichnis definiert wohin die Artefakte abgelegt werden sollen. Die Maven Koordinaten (Gruppen Id, Artefakt Id, Version) sind die von Maven oder Ivy bekannten Parameter:

  • Gruppen Id: Die Gruppen Id ist gewöhnlich eindeutig innerhalb eines Projektes oder einer Organisation.
  • Artefakt Id: Die Artefakt Id ist gewöhnlich der Name des Projektes.
  • Version: Die Version des Artefakts, welches heruntergeladen werden soll. Die Konstanten 'RELEASE' (die letzte Release Version) und 'LATEST' (der letzte Build) sind ebenfalls möglich.
  • Klassifzierung: Hierbei handelt es sich um einen beliebigen Text, der, falls vorhanden, an den Artefaktnamen angehangen wird, unmittelbar nach der Versionsnummer.
  • Erweiterung: Die Erweiterung des Artefakts, welches heruntergeladen wird. Wichtig: dabei handelt es sich nicht immer um die Maven 'Paketierung', obwohl dies im Falle 'jar' gleich ist.
  • Zieldateiname: Soll die Datei nach dem Herunterladen einen anderen Namen erhalten? Hier kann man auch Unterverzeichnisse angeben.
